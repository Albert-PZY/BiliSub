# Acceptance Criteria: B站字幕工作台插件 MVP

**Spec:** `docs/superpowers/specs/2026-05-04-browser-extension-positioning-design.md`
**Date:** 2026-05-05
**Status:** Approved

---

## Criteria

| ID | Description | Test Type | Preconditions | Expected Result |
|----|-------------|-----------|---------------|-----------------|
| AC-001 | 扩展包使用 Manifest V3，并可在 Chrome 稳定版中成功加载且无启动时报错。 | Logic | 准备打包后的扩展目录；在 Chrome 扩展管理页启用开发者模式。 | `manifest_version` 为 `3`；Chrome 成功加载扩展；扩展管理页不显示阻止加载的错误。 |
| AC-002 | 同一份扩展包可在 Edge 稳定版中成功加载且无启动时报错。 | Logic | 使用与 AC-001 相同的扩展目录；在 Edge 扩展管理页启用开发者模式。 | Edge 成功加载扩展；扩展管理页不显示阻止加载的错误。 |
| AC-003 | 扩展权限范围仅覆盖 B 站相关域名与已批准的 LLM 官方域名，不包含本地服务或全站通配权限。 | Logic | 读取打包产物中的 `manifest.json`。 | `host_permissions` 仅包含 `*.bilibili.com/*` 与已批准的 OpenAI、阿里云百炼官方域名；不存在 `<all_urls>`、`http://127.0.0.1/*`、`http://localhost:*/*` 或 Native Messaging 权限。 |
| AC-004 | 在 `https://www.bilibili.com/video/*` 页面中，扩展侧边栏入口可见且可打开工作台。 | UI interaction | 已安装扩展；打开任意匹配路由的 B 站视频页。 | 用户可打开扩展侧边栏；侧边栏渲染工作台主界面，而不是空白页或浏览器默认错误页。 |
| AC-005 | 在非 `bilibili.com/video/*` 页面中，扩展不会启动视频字幕工作流。 | UI interaction | 已安装扩展；打开一个不匹配视频路由的页面。 | 侧边栏若被打开，只显示“当前页面不支持”或等价的非工作态提示；不会发起字幕抓取请求。 |
| AC-006 | 对于有可访问 AI 字幕的 B 站视频，工作台能加载并展示完整字幕列表。 | UI interaction | 浏览器中已处于可访问该视频字幕的登录态；打开一个带 AI 字幕的视频页。 | 侧边栏显示非空字幕列表；每条字幕至少包含时间点与文本；列表总条数大于 0。 |
| AC-007 | 点击字幕列表中的任意一条记录，会将播放器跳转到该条字幕对应的时间点附近。 | UI interaction | 满足 AC-006；播放器已加载完成。 | 点击一条字幕后，播放器当前时间跳转到该条字幕起始时间前后 1 秒范围内。 |
| AC-008 | 关键词搜索会仅显示匹配字幕，并在无匹配时显示明确空状态。 | UI interaction | 满足 AC-006；已知字幕中存在至少一个唯一关键词。 | 输入关键词后，仅显示包含该关键词的字幕条目；输入无匹配关键词时显示空状态提示且列表条数为 0。 |
| AC-009 | “复制全文”会将当前视频完整字幕文本写入剪贴板。 | UI interaction | 满足 AC-006；浏览器允许页面写入剪贴板。 | 点击复制后，剪贴板内容与当前字幕全文完全一致，且不包含 UI 标签或重复段落。 |
| AC-010 | 导出 TXT 会生成可下载的纯文本文件，内容与字幕全文一致。 | UI interaction | 满足 AC-006；浏览器允许下载文件。 | 点击导出 TXT 后生成一个 `.txt` 文件；文件内容与“复制全文”文本完全一致，编码为 UTF-8。 |
| AC-011 | 导出 SRT 会生成可下载的标准 SRT 文件。 | UI interaction | 满足 AC-006；浏览器允许下载文件。 | 点击导出 SRT 后生成一个 `.srt` 文件；文件包含从 `1` 开始连续编号的字幕块，且每个字幕块包含 `HH:MM:SS,mmm --> HH:MM:SS,mmm` 时间轴行。 |
| AC-012 | 当当前视频没有可访问的 AI 字幕时，工作台显示明确错误，并禁用导出和摘要操作。 | UI interaction | 打开一个没有可访问 AI 字幕的视频页。 | 侧边栏显示“无可用 AI 字幕”或等价明确错误；TXT/SRT 导出与摘要按钮均不可点击。 |
| AC-013 | 当用户当前浏览器登录态不足以访问目标字幕时，工作台显示登录或权限错误，并禁用导出和摘要操作。 | UI interaction | 打开一个需要登录态才能访问字幕的视频页；浏览器处于未登录或无权状态。 | 侧边栏显示登录态或权限错误；不会渲染伪造字幕列表；导出与摘要按钮均不可点击。 |
| AC-014 | 设置页只允许选择 `OpenAI` 与 `阿里云百炼` 作为摘要 provider。 | UI interaction | 已安装扩展；打开设置页。 | Provider 选择控件仅包含 `OpenAI` 与 `阿里云百炼` 两个选项。 |
| AC-015 | 设置页不允许用户输入任意自定义 `baseUrl`。 | UI interaction | 打开设置页。 | UI 中不存在可编辑的自定义 `baseUrl` 文本输入框；切换 provider 后仅能使用对应官方域名预设。 |
| AC-016 | `OpenAI` provider 使用固定官方基地址。 | Logic | 在设置页选择 `OpenAI` 并保存配置；检查持久化后的配置对象。 | 持久化配置中的 `baseUrl` 等于 `https://api.openai.com/v1`。 |
| AC-017 | `阿里云百炼` provider 只允许使用官方域名预设。 | Logic | 在设置页选择 `阿里云百炼` 并保存配置；检查持久化后的配置对象。 | 持久化配置中的 `baseUrl` 只能是已内置的阿里云百炼官方域名预设之一。 |
| AC-018 | API Key 仅保存在扩展本地存储，不会写入同步存储。 | Logic | 在设置页输入任意测试 API Key 并保存；读取扩展存储。 | `chrome.storage.local` 中存在保存后的 API Key；`chrome.storage.sync` 中不存在同名字段。 |
| AC-019 | 第一版 API Key 存储不增加额外加密层，保存后可原值读回。 | Logic | 满足 AC-018；从存储层读取保存值。 | 读取出的 API Key 字符串与用户输入值完全一致。 |
| AC-020 | 设置页允许用户填写 `model`，并通过“测试连接”决定摘要功能可用性。 | UI interaction | 打开设置页；分别准备一组有效配置和一组无效配置。 | 有效配置测试成功后显示成功状态；无效配置测试失败后显示错误状态；摘要功能仅在最近一次测试成功后可用。 |
| AC-021 | 摘要请求统一走 OpenAI 兼容的 `POST /v1/chat/completions`。 | UI interaction | 配置任一支持 provider 并触发一次摘要请求；记录扩展网络请求。 | 摘要请求发往所选 provider 官方域名的 `/chat/completions` 路径；不会调用 `/responses` 或任意未批准第三方域名。 |
| AC-022 | 摘要输出固定为简体中文。 | UI interaction | 满足 AC-020；选择一个字幕原文非中文的视频并触发摘要。 | 返回的摘要正文、要点列表和章节小结均为简体中文文本。 |
| AC-023 | 当用户未配置 API Key 时，摘要功能默认不可用。 | UI interaction | 清空摘要设置；打开一个带可访问 AI 字幕的视频页。 | 摘要按钮处于禁用状态，并显示“未配置 API Key”或等价提示。 |
| AC-024 | 当 provider 鉴权失败、限流或服务异常时，摘要不会降级为规则摘要，而是直接失败并保持不可用。 | UI interaction | 满足 AC-020；人为制造一次 `401`、`429` 或 `5xx` 的摘要响应。 | UI 显示对应错误；不会展示本地兜底摘要；摘要按钮在用户修复配置并重新测试成功前保持不可用。 |
| AC-025 | 在未触发摘要功能时，扩展不会向 OpenAI 或阿里云百炼域名发送字幕内容请求。 | UI interaction | 满足 AC-006；打开带字幕视频页，但不执行摘要动作；记录扩展网络请求。 | 网络请求中不存在发往 OpenAI 或阿里云百炼官方域名的字幕或摘要请求。 |
| AC-026 | 在触发一次摘要功能时，扩展只会向当前所选 provider 官方域名发送请求，不会向另一个 provider 或任意其他第三方域名发送摘要请求。 | UI interaction | 满足 AC-020；选择某一 provider 并触发摘要；记录扩展网络请求。 | 本次摘要期间仅出现发往当前所选 provider 官方域名的摘要请求；不存在发往未选中 provider 或其他第三方域名的摘要请求。 |
| AC-027 | 在未安装本地 Python 环境且未运行任何 `localhost` 辅助服务时，公开版扩展仍能完成字幕加载主流程。 | UI interaction | 使用一个未安装本地伴生服务的干净浏览器配置；打开一个满足 AC-006 的视频页。 | 扩展能直接加载字幕列表并执行搜索、复制和导出；网络请求中不存在发往 `127.0.0.1` 或 `localhost` 的依赖性请求。 |
